# PRD: 댓글거울 (Comment Mirror)

## 목표
YouTube 영상 URL 하나만 넣으면 댓글을 자동 분석해서, 크리에이터에게 "잘하는 점 / 개선해야 할 점"을 정리한 한 페이지 리포트를 만들어 준다.

## 사용자
- 본인 채널의 피드백을 빠르게 받고 싶은 1인 YouTube 크리에이터
- 댓글이 수십~수백 개라 수동으로 다 읽기 부담스러운 사람
- "댓글이 좋다/나쁘다" 같은 인상이 아니라 **구체적인 개선 액션**을 원하는 사람

## 핵심 기능
1. **URL 입력 → 리포트 자동 생성**: 영상 URL을 붙여넣으면 댓글 수집 → 분석 → 리포트가 한 번에 만들어진다.
2. **감정 분석 (Sentiment)**: 긍정/중립/부정 비율을 시각적으로 보여준다.
3. **토픽 추출**: 댓글에서 자주 나오는 주제(예: "편집 빠름", "BGM 시끄러움")를 묶어 정리한다.
4. **크리에이터 관점 피드백**:
   - **잘하는 점 (Strengths)**: 시청자가 칭찬하는 포인트
   - **개선점 (Improvements)**: 시청자가 불편해하는 포인트
   - **다음 영상 액션 아이템**: 바로 적용 가능한 구체 제안 3~5개
5. **리포트 히스토리**: LocalStorage에 저장. 사이드바에서 과거 리포트를 다시 열어볼 수 있다.

## 사용 흐름
1. 첫 방문 → 본인 YouTube Data API key + Anthropic API key 1회 입력 (LocalStorage 저장)
2. 메인 화면 URL 입력창에 영상 링크 붙여넣기 → 분석 시작
3. 진행 상태 표시 (댓글 수집 중 → 분석 중)
4. 리포트 페이지로 자동 이동
5. 사이드바에서 과거 리포트 다시 열기 가능

## MVP 제외 사항
- 로그인, 회원가입, 사용자 계정
- 서버, 데이터베이스 (모든 데이터는 브라우저 LocalStorage)
- 채널 전체 분석 (영상 단위만)
- 답글(reply) 분석 (top-level 댓글만)
- 자동 번역 (댓글 언어 그대로 분석)
- 결제, 유료 플랜
- 리포트 공유 링크 (다운로드/스크린샷으로 대체)

## 디자인 방향
- 다크모드 고정, 무채색 베이스
- 도구 느낌 — 마케팅 페이지가 아니라 매일 쓰는 대시보드
- 포인트 색상은 데이터 의미에만 사용 (긍정 녹색, 부정 적색)
- 차트/그래프는 최소화. 숫자와 텍스트가 주인공
